¿Qué es JavaScript?

Descripción

JavaScript es un lenguaje de programación interpretado, orientado a objetos y multiplataforma. Es uno de los tres lenguajes fundamentales de la web, junto con HTML y CSS.

Historia y evolución de JavaScript

Descripción

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape. Desde entonces, ha evolucionado significativamente, pasando de ser un simple lenguaje de scripting a una poderosa herramienta para el desarrollo web moderno.

Uso de JavaScript en desarrollo web

Descripción

JavaScript se utiliza principalmente para crear interactividad en las páginas web, pero también se ha expandido a otros ámbitos como desarrollo de servidores, aplicaciones móviles y de escritorio.

Herramientas necesarias para JavaScript

Descripción

Para comenzar a programar en JavaScript, necesitarás un editor de código, un navegador web y las herramientas de desarrollo del navegador. También aprenderemos sobre Node.js y npm para el desarrollo backend.

Primer programa: Hola mundo

Descripción

Crearemos nuestro primer programa en JavaScript, aprendiendo a incluir código JavaScript en una página HTML y a mostrar mensajes en la consola y en la página web.

Variables en JavaScript

Descripción

Aprenderemos sobre las variables en JavaScript, incluyendo las diferencias entre var, let y const, y cómo declarar y usar variables correctamente.

Tipos de datos en JavaScript

Descripción

Exploraremos los diferentes tipos de datos en JavaScript: números, strings, booleanos, null, undefined, objetos y arrays.

Operadores en JavaScript

Descripción

Aprenderemos sobre los diferentes tipos de operadores en JavaScript: aritméticos, de asignación, de comparación, lógicos y más.

Actividad: Fundamentos de JavaScript

Descripción

Practicaremos los conceptos aprendidos en el módulo de fundamentos con ejercicios prácticos y problemas para resolver.

Sentencias condicionales en JavaScript

Descripción

Aprenderemos a usar las sentencias if, else y else if para controlar el flujo de nuestro programa basado en condiciones. Veremos ejemplos prácticos de cómo implementar lógica condicional en JavaScript.

Operador ternario en JavaScript

Descripción

Aprenderemos a usar el operador ternario como una forma concisa de escribir condicionales simples en JavaScript.

Sentencia switch en JavaScript

Descripción

Exploraremos la sentencia switch como una alternativa a múltiples if-else cuando necesitamos comparar una variable con múltiples valores.

Ciclos en JavaScript

Descripción

Aprenderemos a usar los diferentes tipos de bucles en JavaScript: for, while, do...while y for...of, y cuándo usar cada uno.

Actividad: Estructuras de control

Funciones en JavaScript

Descripción

Aprenderemos a declarar y usar funciones en JavaScript, incluyendo funciones declaradas, expresiones de función y funciones anónimas.

Parámetros y retorno de funciones

Descripción

Exploraremos cómo pasar parámetros a funciones y cómo devolver valores, incluyendo el uso de valores por defecto y el operador rest.

Funciones flecha en JavaScript

Descripción

Aprenderemos sobre las funciones flecha (arrow functions), una sintaxis más concisa para escribir funciones en JavaScript moderno.

Scope en JavaScript

Descripción

Entenderemos el concepto de scope (alcance) en JavaScript, incluyendo scope global, local y el comportamiento de las variables con var, let y const.

Descripción

Bucles con arrays

Manipulación del DOM

Descripción

Eventos en JavaScript

Descripción

Aprenderemos a manejar eventos del usuario como clicks, teclas presionadas y cambios en formularios.

Asincronía en JavaScript

Callbacks en JavaScript

Descripción

Exploraremos el uso de callbacks para manejar operaciones asíncronas y sus limitaciones.

Promesas en JavaScript

Descripción

Aprenderemos a usar promesas para manejar operaciones asíncronas de manera más elegante y mantenible.

Async/Await en JavaScript

Descripción

Exploraremos la sintaxis async/await como una forma más moderna y legible de trabajar con código asíncrono.

Quiz del Curso

Evaluación

1. ¿Qué es un algoritmo?

  • Un lenguaje de programación
  • Un conjunto de instrucciones para resolver un problema
  • Un tipo de variable
  • Un error en el código

2. ¿Cuál es la diferencia entre pseudocódigo y un lenguaje de programación?

  • El pseudocódigo es más difícil de entender
  • El pseudocódigo es independiente de cualquier lenguaje de programación
  • El pseudocódigo no puede ser ejecutado
  • El pseudocódigo es más rápido de escribir

3. ¿Qué representa un diagrama de flujo?

  • La estructura de una base de datos
  • El diseño de una interfaz de usuario
  • El flujo de un proceso o algoritmo
  • La arquitectura de un sistema
